Hand written number recognition using TensorFlow and JavaFX

This project is about building a handwritten number recognition system using TensorFlow and JavaFX. The system uses a deep learning model trained on the MNIST dataset to recognize handwritten digits and a JavaFX user interface to allow users to draw digits and see the predicted results.

    Details

    Overview

    The project involves the following steps:

    • Data Preprocessing - The MNIST dataset is preprocessed using Python - TensorFlow and Numpy to normalize the pixel values and prepare it for model training.
    • Model Training and Testing - A deep learning model is built using TensorFlow and trained on the preprocessed MNIST dataset. The model is evaluated based on its accuracy in recognizing handwritten digits.
    • User Interface Development - A user interface is developed using JavaFX to allow users to draw digits and see predicted results. User interface communicates with trained TensorFlow model to make predictions.
    • Model Deployment - The trained TensorFlow model and the JavaFX user interface are packaged into a standalone application that can be deployed and run on any computer.
