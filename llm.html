Cricket Statistics LLM Chatbot

The Cricket Statistics LLM Chatbot is an AI-powered tool designed to help users retrieve cricket-related statistical insights by converting natural language queries into executable Python code. The chatbot leverages a large language model (LLM), specifically Google Gemini Pro, to generate accurate and efficient Pandas code for analyzing cricket data stored in multiple DataFrames. The main features of the project include:

  • Dataset: The project uses four cricket-specific datasets:
    • batters_against_bowlingtype.xlsx (df1): Performance of batsmen against various bowling styles.
    • Bowling_against_left_right_handers.xlsx (df2): Bowlers performance against left-handed and right-handed batsmen.
    • batting_stats.csv (df3): Overview of batsmens performance across various matches.
    • bowling_stats.csv (df4): Breakdown of bowlers performance across multiple matches.
  • Prompt Handling: Users input natural language prompts, which the LLM converts into Pandas code to filter, sort, and aggregate cricket statistics.
  • Execution & Results: The generated code is executed on the relevant DataFrames, and the results are either displayed as tables or visualized using matplotlib.
  • Feedback Mechanism: A feedback system allows users to provide a thumbs up or thumbs down for the generated results. Feedback, along with the prompt and generated code, is stored in an SQLite database, facilitating model improvements and fine-tuning.

This project offers an innovative approach to accessing cricket statistics, simplifying complex queries through natural language interaction.

Tech Stack
  • Front-End - Streamlit, Python
  • LLMs Used - Google Gemini, GPT
  • Libraries - Pandas, Matplotlib
